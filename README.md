# homebrew-fonts

Personal [Homebrew](https://brew.sh) tap for fonts.

```sh
brew tap unship/fonts
brew install --cask font-noto-color-emoji-cjk
```

## Casks

- **font-noto-color-emoji-cjk** — colour emoji + symbols pre-fit to the CJK cell,
  for grid-aligned editors (especially Emacs).
  Source: https://github.com/unship/noto-color-emoji-cjk
